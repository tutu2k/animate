一、选择题（共15分）

第1题（1分）
下列粒子中，带负电的是（ ）
A. 电子
B. 质子
C. 中子
D. 原子核
答案：A
解析：
电子带负电，质子带正电，中子不带电，原子核由质子和中子组成，带正电。
因此选 A. 电子
第2题（1分）
"请勿大声喧哗"中的"大声"指的是声音的（ ）
A. 音调
B. 响度
C. 音色
D. 频率
答案：B
解析：
"大声"指的是声音的大小，即响度。
音调指声音的高低，由频率决定；音色是声音的特色。
因此选 B. 响度
第3题（1分）
月球是地球的（ ）
A. 恒星
B. 行星
C. 卫星
D. 彗星
答案：C
解析：
月球围绕地球运转，是地球的天然卫星。
太阳是恒星，地球是行星。
因此选 C. 卫星
第4题（1分）
光从空气斜射入水中时，折射角（ ）入射角。
A. 大于
B. 等于
C. 小于
D. 无法确定
答案：C
解析：
光从空气（光疏介质）斜射入水（光密介质）时，折射光线向法线偏折，折射角小于入射角。
反之，从水射入空气时，折射角大于入射角。
因此选 C. 小于
第5题（2分）
甲、乙两车同时同地向东做直线运动，它们的s-t图像如图所示。下列说法正确的是（ ）
A. 甲的速度大于乙的速度
B. 以甲为参照物，乙向东运动
C. 经过6秒，甲、乙相距1.2米
D. 甲、乙都做变速直线运动
答案：C
解析：
从s-t图像可知：
• 甲、乙都是匀速直线运动（图像为直线）
• v甲 = 0.2 m/s，v乙 = 0.4 m/s，所以乙的速度大
• 以甲为参照物，乙向西运动（乙快于甲）
• 6秒时：s甲 = 1.2m，s乙 = 2.4m，相距 = 2.4 - 1.2 = 1.2m
因此选 C

二、填空题（共20分）

第6题（2分）
上海地区家庭电路的电压为______伏，冰箱与空调是______连接的（选填"串联"或"并联"）。
答案：220；并联
解析：
• 我国家庭电路标准电压为 220V
• 家庭电路中各用电器采用 并联 连接，互不影响，都能获得220V电压
第7题（2分）
某导体两端电压为6伏，通过它的电流为0.3安，该导体的电阻为______欧；当它两端电压变为12伏时，通过它的电流为______安。
答案：20；0.6
解析：
根据欧姆定律 I = U/R：
R = U/I = 6/0.3 = 20Ω
电阻是导体本身的性质，与电压无关，仍为20Ω。
当U = 12V时：
I = U/R = 12/20 = 0.6A
第8题（2分）
质量为2千克的水温度升高了10℃，吸收的热量为______焦。[c水 = 4.2×10³ J/(kg·℃)]
答案：8.4×10⁴
解析：
根据吸热公式 Q = cmΔt：
Q = 4.2×10³ × 2 × 10 = 8.4×10⁴ J
第9题（2分）
重为10牛的物体浸没在水中，排开水的体积为5×10⁻⁴米³，物体受到的浮力为______牛。若增大物体在水中的深度，它受到的浮力将______（选填"变大"、"变小"或"不变"）。
答案：4.9；不变
解析：
根据阿基米德原理 F浮 = ρ液gV排：
F浮 = 1.0×10³ × 9.8 × 5×10⁻⁴ = 4.9N
物体已浸没，V排不变，所以浮力 不变（与深度无关）
第10题（3分）
在如图所示的电路中，电源电压保持不变。闭合开关S，电路正常工作。一段时间后，发现其中一个电表示数变大，另一个电表示数变小。已知电路中只有一处故障，且故障只发生在电阻R₁或R₂上。

（1）若电压表V示数变大，电流表A示数变小，则故障可能是______。
（2）若电压表V示数变小，电流表A示数变大，则故障可能是______。
答案：（1）R₁断路；（2）R₁短路
解析：
这是典型的串联电路故障分析题。

（1）V变大，A变小 → 说明总电阻变大 → R₁断路
此时电压表测电源电压，示数变大；电路断开，电流为0。

（2）V变小，A变大 → 说明总电阻变小 → R₁短路
此时电压表被短路，示数为0；只有R₂工作，电流变大。

三、作图题（共5分）

第11题（2分）
在图中，根据平面镜成像特点，画出物体AB在平面镜中所成的像A'B'。
作图要点
作图步骤：
1. 从A、B两点分别向镜面作垂线
2. 延长垂线，使像距等于物距
3. 连接A'、B'，用虚线表示
4. 标注像点A'、B'

注意：像与物关于镜面对称，用虚线画，标垂直符号
第12题（3分）
在图中，重为6牛的物体静止在水平地面上，请用力的图示法画出物体对地面的压力F。
作图要点
作图步骤：
1. 确定作用点：在接触面中心
2. 确定方向：竖直向下
3. 确定大小：F = G = 6N
4. 选定标度（如1cm = 2N）
5. 画线段并标注箭头和F=6N

注意：压力作用在接触面上，不是重心；要标注标度

四、计算题（共20分）

第13题（6分）
质量为5千克的物体放在水平地面上，它与地面的接触面积为0.01米²。求：
（1）物体的重力G；
（2）物体对地面的压强p。（g = 10 N/kg）
参考答案
已知：m = 5kg，S = 0.01m²，g = 10N/kg

解：
（1）物体的重力：
G = mg = 5 × 10 = 50N
（2）物体对地面的压力：F = G = 50N
物体对地面的压强：
p = F/S = 50/0.01 = 5000Pa
答：物体的重力为50N，对地面的压强为5000Pa。
第14题（7分）
在如图所示的电路中，电源电压为12伏且保持不变，电阻R₁的阻值为10欧。闭合开关S后，电流表的示数为0.8安。求：
（1）电阻R₁两端的电压U₁；
（2）电阻R₂的阻值；
（3）通电10秒，电流通过R₂所做的功W₂。
参考答案
已知：U = 12V，R₁ = 10Ω，I = 0.8A，t = 10s

解：
（1）电阻R₁两端的电压：
U₁ = IR₁ = 0.8 × 10 = 8V
（2）电阻R₂两端的电压：
U₂ = U - U₁ = 12 - 8 = 4V
电阻R₂的阻值：
R₂ = U₂/I = 4/0.8 = 5Ω
（3）电流通过R₂所做的功：
W₂ = U₂It = 4 × 0.8 × 10 = 32J
答：R₁两端电压为8V，R₂阻值为5Ω，电流通过R₂做功32J。
第15题（7分）
如图所示，轻质杠杆OA可绕O点转动，在A端挂一重为20牛的物体，在B点施加一个竖直向上的力F，使杠杆在水平位置平衡。已知OA = 0.6米，OB = 0.4米。求：
（1）力F的大小；
（2）若将力F的方向改为竖直向下，要使杠杆仍在水平位置平衡，力F的作用点应向______移动（选填"左"或"右"）。
参考答案
已知：G = 20N，OA = 0.6m，OB = 0.4m

解：
（1）根据杠杆平衡条件 F₁L₁ = F₂L₂：
F × OB = G × OA
F × 0.4 = 20 × 0.6
F = 30N
（2）若F改为竖直向下，要使杠杆平衡，需要增大力臂，所以应向 右 移动。

答：力F的大小为30N，应向右移动。

五、实验题（共10分）

第16题（4分）
小明同学在做"探究凸透镜成像规律"的实验时，所用凸透镜的焦距为10厘米。

（1）实验前，应调整烛焰、凸透镜和光屏的中心大致在______。
（2）当蜡烛距凸透镜15厘米时，移动光屏，可在光屏上得到一个倒立、______的实像（选填"放大"、"缩小"或"等大"）。
（3）若将蜡烛远离凸透镜，要使光屏上仍能得到清晰的像，光屏应向______移动（选填"靠近"或"远离"）凸透镜。
答案
答案：
（1）同一高度（使像成在光屏中央）
（2）f < u < 2f，成 放大 的倒立实像
（3）物远像近像变小，光屏应向 靠近 凸透镜方向移动

凸透镜成像规律总结：
• u > 2f：倒立缩小实像（照相机）
• f < u < 2f：倒立放大实像（投影仪）
• u < f：正立放大虚像（放大镜）
第17题（6分）
小红同学在做"测定小灯泡电功率"的实验时，所用小灯泡的额定电压为2.5伏。实验中，她正确连接电路，闭合开关后，发现小灯泡不亮，电流表无示数，电压表有示数且接近电源电压。

（1）电路故障可能是______。
（2）排除故障后，调节滑动变阻器，使小灯泡正常发光，此时电流表的示数为0.3安，则小灯泡的额定功率为______瓦。
（3）若继续调节滑动变阻器，使小灯泡两端电压为2伏，则小灯泡的实际功率______额定功率（选填"大于"、"小于"或"等于"）。
答案
答案与解析：
（1）电流表无示数→断路；电压表有示数→电压表与电源连通
所以故障是：小灯泡断路

（2）小灯泡正常发光时 U = 2.5V，I = 0.3A
P额 = UI = 2.5 × 0.3 = 0.75W
（3）U实 = 2V < U额 = 2.5V，所以 P实 小于 P额

故障分析技巧：
• 电流表无示数+电压表有示数→被测用电器断路
• 电流表有示数+电压表无示数→被测用电器短路

第1题（1分）
一个鸡蛋的质量约为（ ）
A. 5克
B. 50克
C. 500克
D. 5千克
答案：B
解析：
两个鸡蛋约重1牛，即质量约100克，所以一个鸡蛋约 50克。
这是常见的生活常识估算题。
第5题（2分）
甲、乙两物体同时从P点出发，沿同一直线相向运动，它们的s-t图像如图所示。已知乙距离P点1米，下列说法正确的是（ ）
A. 甲的速度一定大于乙的速度
B. 经过3秒，甲、乙一定相遇
C. 乙距离P点1米的位置有2种可能
D. 以甲为参照物，乙一定静止
答案：C
解析：
本题关键是"乙距离P点1米"有两种情况：
• 乙在P点左侧1米处
• 乙在P点右侧1米处
所以选 C

易错点：s-t图像中，同一距离值可能对应两个位置，要注意多解性

二、填空题（部分）

第7题（2分）
家庭电路中，电灯与控制它的开关是______连接的，电灯与电视机是______连接的。（均选填"串联"或"并联"）
答案：串联；并联
解析：
• 开关必须与被控制的用电器 串联
• 各用电器之间采用 并联，互不影响
第12题（3分）
在如图所示的电路中，电源电压保持不变。闭合开关S，电路正常工作。若电阻R₁发生故障，请根据电表示数变化判断故障类型：

（1）若电压表V示数为零，电流表A示数变大，则R₁______。
（2）若电压表V示数变大，电流表A示数为零，则R₁______。
答案：（1）短路；（2）断路
解析：
（1）V=0，A变大 → R₁被短路 → R₁短路
（2）V变大，A=0 → 电路断开 → R₁断路

这是经典的串联电路故障判断题。

三、计算题（部分）

第14题（7分）
质量为2千克的铝块温度从30℃升高到80℃，求铝块吸收的热量Q吸。[c铝 = 0.9×10³ J/(kg·℃)]
参考答案
已知：m = 2kg，t₀ = 30℃，t = 80℃，c铝 = 0.9×10³ J/(kg·℃)

解：
Q吸 = cm(t - t₀)
= 0.9×10³ × 2 × (80 - 30)
= 0.9×10³ × 2 × 50
= 9×10⁴ J
答：铝块吸收的热量为9×10⁴焦。
第15题（7分）
如图所示，圆柱体甲和乙放置在水平地面上。已知甲的质量为6千克，底面积为0.02米²；乙的质量为4千克，底面积为0.01米²。求：
（1）甲对地面的压强p甲；
（2）若在甲、乙上方分别放置质量相等的物体后，它们对地面的压强相等，求所放物体的质量m。
参考答案
已知：m甲=6kg，S甲=0.02m²，m乙=4kg，S乙=0.01m²，g=10N/kg

解：
（1）甲对地面的压力：F甲 = G甲 = m甲g = 6×10 = 60N
p甲 = F甲/S甲 = 60/0.02 = 3000Pa
（2）设所放物体质量为m，则：
p甲' = p乙'
(60 + mg)/0.02 = (40 + mg)/0.01
解得：m = 2kg
答：甲对地面压强为3000Pa，所放物体质量为2kg。

第3题（1分）
下列过程中，动能转化为重力势能的是（ ）
A. 苹果自由下落
B. 火箭加速上升
C. 滚摆向上运动

答案：C

• B：化学能→机械能

因此选 C

某导体两端的电压为3伏，通过它的电流为0.2安，10秒内通过该导体横截面的电荷量为______库，该导体的电阻为______欧。

Q = It = 0.2 × 10 = 2C

R = U/I = 3/0.2 = 15Ω

三、情景题

第13题（4分）
冬天水管容易冻裂，这是因为水结冰后体积会______（选填"变大"、"变小"或"不变"）。已知冰的密度为0.9×10³ kg/m³，水的密度为1.0×10³ kg/m³，则1立方米的水结成冰后，体积变为______立方米。
答案：变大；1.11（或10/9）
解析：
（1）水结冰后密度变小，质量不变，由ρ=m/V可知体积 变大。

（2）水的质量：
m = ρ水V水 = 1.0×10³ × 1 = 1000kg
冰的体积：
V冰 = m/ρ冰 = 1000/(0.9×10³) = 10/9 ≈ 1.11 m³
生活应用：这就是为什么冬天水管要保温，防止水结冰体积膨胀导致破裂。
2022年试卷点评：试卷总体难度不大，注重基础，缺少了往年的压强和动态电路选择题，反而让一些学生不适应。情景题贴近生活，体现新课标要求。
