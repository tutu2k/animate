📚 上海物理中考题目整理

核心考点 · 重要公式 · 典型例题

1. 运动和力

核心知识点

  • 速度公式：v = s/t
  • 牛顿第一定律（惯性定律）
  • 二力平衡条件
  • 摩擦力的大小与方向
速度公式：v = s / t
单位换算：1 m/s = 3.6 km/h

📝 典型例题

一辆汽车以20m/s的速度匀速行驶，经过5s后通过的路程是多少？

解：
已知：v = 20 m/s，t = 5 s
由 v = s/t 得：
s = v × t = 20 × 5 = 100 m

2. 压强

压强公式：p = F / S
液体压强：p = ρgh
单位：1 Pa = 1 N/m²

注意事项

  • 固体压强：先求压力F，再求压强p
  • 液体压强：先用p=ρgh求压强，再用F=pS求压力
  • 受力面积S是接触面积，不是物体表面积

📝 典型例题

一个质量为60kg的人站立时，双脚与地面的接触面积为400cm²，求人对地面的压强。（g=10N/kg）

解：
人的重力：G = mg = 60 × 10 = 600 N
人对地面的压力：F = G = 600 N
受力面积：S = 400 cm² = 0.04 m²
压强：p = F/S = 600/0.04 = 15000 Pa

3. 浮力

阿基米德原理：F浮 = ρ液gV排
称重法：F浮 = G - F示
漂浮条件：F浮 = G物
物体状态 浮力与重力关系 密度关系
漂浮 F浮 = G ρ物 < ρ液
悬浮 F浮 = G ρ物 = ρ液
下沉 F浮 < G ρ物 > ρ液

4. 简单机械

杠杆平衡条件：F₁L₁ = F₂L₂
滑轮组：F = (G物 + G动) / n
机械效率：η = W有 / W总 × 100%

滑轮组要点

  • n为承担物重的绳子段数
  • s = nh（绳子自由端移动距离）
  • v绳 = nv物（绳子自由端速度）

1. 温度与热量

吸热公式：Q吸 = cm(t - t₀)
放热公式：Q放 = cm(t₀ - t)
燃料燃烧：Q = mq 或 Q = Vq

比热容

  • 水的比热容：c水 = 4.2×10³ J/(kg·℃)
  • 比热容是物质的特性，与质量、温度变化无关
  • 水的比热容大，常用于冷却剂和取暖介质

📝 典型例题

将2kg的水从20℃加热到70℃，需要吸收多少热量？

解：
Q吸 = cm(t - t₀)
= 4.2×10³ × 2 × (70 - 20)
= 4.2×10³ × 2 × 50
= 4.2×10⁵ J

2. 物态变化

变化名称 过程 吸/放热 实例
熔化 固→液 吸热 冰化成水
凝固 液→固 放热 水结成冰
汽化 液→气 吸热 水蒸发
液化 气→液 放热 露水形成
升华 固→气 吸热 干冰升华
凝华 气→固 放热 霜的形成

3. 内能与热机

改变内能的方式

  • 做功：机械能 ↔ 内能（如摩擦生热）
  • 热传递：内能的转移（如加热物体）
热机效率：η = W有 / Q放 × 100%
四冲程汽油机：吸气→压缩→做功→排气

1. 光的直线传播

现象与应用

  • 影子的形成
  • 日食、月食
  • 小孔成像（倒立实像）
  • 激光准直
光速：c = 3×10⁸ m/s（真空中）
光年：长度单位，光在一年中传播的距离

2. 光的反射

反射定律

  • 反射光线、入射光线、法线在同一平面内
  • 反射光线和入射光线分居法线两侧
  • 反射角 = 入射角
类型 特点 应用
镜面反射 平行光反射后仍平行 平面镜、水面
漫反射 平行光反射后向各个方向 能从各方向看到物体
平面镜成像特点：
• 像与物大小相等
• 像与物到镜面距离相等
• 像与物的连线与镜面垂直
• 成虚像

3. 光的折射

折射规律

  • 光从空气斜射入水中：折射角 < 入射角
  • 光从水斜射入空气中：折射角 > 入射角
  • 垂直入射时，传播方向不变

📝 常见现象

  • 筷子在水中"弯折"
  • 池水看起来变浅
  • 海市蜃楼
  • 彩虹的形成

4. 透镜及其应用

透镜类型 对光线作用 应用
凸透镜 会聚 放大镜、照相机、投影仪
凹透镜 发散 近视眼镜
凸透镜成像规律：
u > 2f：倒立缩小实像（照相机）
f < u < 2f：倒立放大实像（投影仪）
u < f：正立放大虚像（放大镜）

1. 电流、电压、电阻

欧姆定律：I = U / R
变形公式：U = IR，R = U/I
物理量 符号 单位 测量工具
电流 I 安培(A) 电流表（串联）
电压 U 伏特(V) 电压表（并联）
电阻 R 欧姆(Ω) 欧姆表/伏安法

串并联电路特点

  • 串联：I处处相等，U总=U₁+U₂，R总=R₁+R₂
  • 并联：U处处相等，I总=I₁+I₂，1/R总=1/R₁+1/R₂

2. 电功与电功率

电功：W = UIt = Pt
电功率：P = UI = W/t = I²R = U²/R
单位：1 kW·h = 3.6×10⁶ J

📝 典型例题

一只标有"220V 100W"的灯泡正常工作时的电流和电阻是多少？

解：
由 P = UI 得：
I = P/U = 100/220 ≈ 0.45 A

由 P = U²/R 得：
R = U²/P = 220²/100 = 484 Ω

3. 焦耳定律

焦耳定律：Q = I²Rt
（适用于所有电路的电热计算）

纯电阻电路

  • 电能全部转化为内能：W = Q
  • 可用公式：Q = I²Rt = UIt = U²t/R

📝 典型例题

一根电阻丝阻值为10Ω，通过的电流为2A，通电5min产生的热量是多少？

解：
t = 5 min = 300 s
Q = I²Rt = 2² × 10 × 300 = 12000 J

4. 家庭电路与安全用电

安全用电原则

  • 不接触低压带电体，不靠近高压带电体
  • 更换灯泡、搬动电器前应断开电源开关
  • 保险丝熔断后不能用铜丝、铁丝代替
  • 有金属外壳的用电器要接地线
组成部分 作用
电能表 测量消耗的电能
总开关 控制整个电路的通断
保险丝/空气开关 电流过大时自动切断电路
插座 左零右火上接地

1. 机械能

动能：Eₖ = ½mv²
重力势能：Eₚ = mgh
机械能：E = Eₖ + Eₚ

影响因素

  • 动能：与质量和速度有关（速度影响更大）
  • 重力势能：与质量和高度有关
  • 弹性势能：与弹性形变程度有关

📝 能量转化实例

  • 滚摆：动能 ↔ 重力势能
  • 弹簧振子：动能 ↔ 弹性势能
  • 卫星绕地球：近地点动能大，远地点势能大

2. 功和功率

功：W = Fs（力和位移方向一致）
功率：P = W/t = Fv
单位：1 W = 1 J/s

做功的两个必要因素

  • 作用在物体上的力
  • 物体在力的方向上通过的距离

📝 不做功的情况

  • 有力无距离（推墙不动）
  • 有距离无力（惯性运动）
  • 力与距离垂直（水平运动时重力不做功）

3. 能量守恒定律

核心内容

  • 能量既不会凭空产生，也不会凭空消失
  • 它只会从一种形式转化为另一种形式
  • 或者从一个物体转移到另一个物体
  • 在转化和转移过程中，能量的总量保持不变
能量守恒表达式：
E初 = E末
或 ΔE增 = ΔE减
